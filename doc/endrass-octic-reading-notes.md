# Endraß Octic 阅读笔记

## 论文

S. Endraß, "A Projective Surface of Degree Eight with 168 Nodes", arXiv:alg-geom/9507011.

本地文件：

- `arxiv/endrass_9507011.pdf`
- `arxiv/endrass_9507011.tex`
- `arxiv/endrass_9507011.txt`

## 目标问题

记 `mu(d)` 为复射影三空间 `P^3` 中、次数为 `d` 且只有 ordinary double points/nodes 的代数曲面最多能有多少个节点。

Endraß 的结果是：

```text
168 <= mu(8) <= 174
```

其中 `168` 来自一个显式八次曲面构造；`174` 来自 Miyaoka 上界：

```text
mu(d) <= floor((4/9) * d * (d - 1)^2)
```

代入 `d = 8`：

```text
(4/9) * 8 * 7^2 = 1568/9 = 174.222...
```

节点数是整数，所以 `mu(8) <= 174`。

备注：SURFER 的说明文字把这个上界归给 Varchenko；并行核对后，`d=8` 时 Varchenko/Arnold number 给出的上界是 `180`，不是 `174`。Endraß 本文的 `174` 明确应归给 Miyaoka。

## 方程框架

论文从一个 `D_8` 对称的八次曲面族开始：

```text
F = P - Q
X = { F = 0 }
```

其中

```text
P = product_{j=0}^{7} (cos(j*pi/4) x + sin(j*pi/4) y - w)
```

等价写成

```text
P = 1/4 (x^2 - w^2)(y^2 - w^2)((x+y)^2 - 2w^2)((x-y)^2 - 2w^2)
```

`P` 是八个平面的乘积，这八个平面围出正八边形方向。它负责 `D_8` 对称性。

`Q` 是一个四次多项式的平方：

```text
Q = (
  a(x^2+y^2)^2
  +(x^2+y^2)(b z^2 + c z w + d w^2)
  +e z^4 + f z^3 w + g z^2 w^2 + h z w^3 + i w^4
)^2
```

因此 `F` 是八次多项式。

## 为什么天然有 112 个节点

`P` 在八个平面 `H_j` 上为零。任意两个平面相交得到一条线，共有：

```text
C(8, 2) = 28
```

条交线。

在这些交线上，`P` 至少二阶为零。`Q` 是四次曲面的平方，所以在四次曲面与这些交线的交点处也二阶为零。

一般情况下，每条线和四次曲面相交 4 次，因此得到：

```text
28 * 4 = 112
```

个二重点。对一般参数，它们是 nodes。

这是构造的底盘：不是先找 168 个点，而是先用八个平面乘积和平方结构自动制造 112 个候选节点。

## 对称性如何减少搜索

曲面有 `D_8` 对称。论文还进一步设：

```text
c = f = h = 0
```

使曲面对 `z -> -z` 也对称，即有 `D_8 x Z_2` 对称。

于是只需要研究两个代表性反射平面：

```text
E_0 = { y = 0 }
E_1 = { x = (1 + sqrt(2)) y }
```

其他平面上的情况由对称群轨道推出。

## 额外 56 个节点怎么来

论文将问题降到两个平面截线：

```text
C_0 = X ∩ E_0
C_1 = X ∩ E_1
```

再通过 Segre trick 研究相应的平面四次曲线 `C~_0` 和 `C~_1`。

关键轨道计数：

- 一般位置的节点会在三维曲面上诱导长度 `16` 的奇点轨道。
- 坐标轴上的接触点会诱导长度 `8` 的奇点轨道。

最终特殊参数让平面曲线额外出现：

```text
s_3: 一个额外节点          -> 16 个节点
t_3: 一个额外接触点        -> 8 个节点
u_5: 一个额外节点          -> 16 个节点
v_1: 一个额外接触点        -> 8 个节点
v_2: 一个额外接触点        -> 8 个节点
```

合计：

```text
16 + 8 + 16 + 8 + 8 = 56
```

所以总节点数：

```text
112 + 56 = 168
```

## 最终参数

论文最终取：

```text
a = -1/4 * (1 + sqrt(2))
b =  1/2 * (2 + sqrt(2))
d =  1/8 * (2 + 7 sqrt(2))
g =  1/2 * (1 - 2 sqrt(2))
i = -1/16 * (1 + 12 sqrt(2))
```

并且：

```text
c = f = h = 0
e = -1
```

你给 SURFER 的仿射方程基本就是把论文的射影方程取 `w = 1` 后再整体缩放得到的版本。

## 节点验证原则

对曲面 `F = 0`，奇点满足：

```text
F = 0
dF/dx = 0
dF/dy = 0
dF/dz = 0
```

如果该点 Hessian 矩阵行列式非零，则这个奇点是 ordinary double point，也就是 node。

论文中 Endraß 用 Maple 计算 Hessian 行列式，证明列出的候选奇点确实是 nodes。

## 还需要继续理解的点

1. `Segre trick` 在这里如何把平面四次曲线的节点/接触点提升到三维八次曲面的节点轨道。
2. Endraß 参数搜索的实际过程：论文给了最终参数和中间代换，但没有公开完整 Maple 搜索脚本。
3. “无其他奇点”的证明细节：论文说在八个反射平面上用 Maple 检查，平面外则用一个 `D_n` 对称曲面的轨道排除引理。
4. Miyaoka 上界的原始证明不在本文中，只被引用；Varchenko 是另一条 spectrum 半连续性上界路线，`d=8` 时较弱。

## 本地复现建议，暂不写代码

第一阶段：符号理解

- 验证 `P` 的乘积形式等价于四个二次因子的形式。
- 在 `w = 1` 的仿射图中重写 Endraß 方程。
- 明确 `112` 个基础节点来自 `H_j ∩ H_k` 与四次曲面的交点。

第二阶段：小模型练习

- 先用较低次数或较少对称平面做 toy model。
- 练习求解 `F = grad F = 0`。
- 练习用 Hessian 判断 node。

第三阶段：复现 Endraß octic

- 先用 CAS 或符号库验证论文列出的候选点。
- 再用数值方法扫出奇点，验证轨道计数。
- 最后考虑 Rust 实现：Rust 适合精确代数数、数值求解、轨道枚举和可视化管线；Groebner basis/消元若要严肃做，可能需要调用外部 CAS 或先实现一个小范围专用版本。
