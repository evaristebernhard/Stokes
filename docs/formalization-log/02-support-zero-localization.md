# 02. 支撑与 zero-localization：为什么普通 chart representative 会骗人

## 数学阻塞

纸面证明里，局部 chart representative 总是默认只在 chart domain 上讨论。例如数学家会写：

```text
φ_* (ρ ω) is supported in Q
```

这里的意思其实是：在 chart 有定义、transition 有意义的区域内，局部化后的 form 的坐标表达支撑落在选定盒子里。

Lean 中函数是 total 的。`transitionPullbackInChart` 作为一个函数必须对所有坐标点都有值，即使该点不在 source chart、target chart 或 overlap 内。于是一个看似自然的命题：

```lean
tsupport (ManifoldForm.transitionPullbackInChart I x0 x1 η) ⊆ halfSpaceSupportBox a b
```

可能太强，也可能根本不是纸面证明想表达的命题。它把 chart 外的 totalized 值也纳入 support 判断。这个问题非常隐蔽，因为数学家默认“chart 外不讨论”，而 Lean 必须讨论。

## 错误或不自然路线

早期 represented Stokes 路线里，很多 local support 字段会要求 ordinary total representative 的支撑落进 coordinate support 或 half-space box。这样的字段会把真正的数学问题扭曲成一个 totalization 问题。

这导致两种坏后果：

1. theorem input 变大：用户或上层 constructor 必须手填 `base_tsupport_subset_coordSupport` 之类证书。
2. theorem shape 失真：命题看上去像是关于局部 form 支撑，实际上在要求 totalized chart function 在 chart 外也表现良好。

后来路线明确转向 zero/localized representative。核心判断是：不要证明一个假的 total support theorem，而要定义一个在 chart 外强制为零的 representative。

## 最终解决方案

基础文件：

```text
Stokes/ManifoldFormZero.lean
```

关键 API：

```lean
zeroOutside
support_zeroOutside_subset
ManifoldForm.transitionPullbackInChartZero
ManifoldForm.transitionPullbackInChartZero_support_subset_source
ManifoldForm.transitionPullbackInChartZero_support_subset_target
ManifoldForm.transitionPullbackInChartZero_support_subset_overlap
ManifoldForm.transitionPullbackInChartZero_eqOn_transitionPullbackInChart
```

这些 theorem 的作用是把“chart 外不讨论”变成“chart 外为零”。这样 support 命题重新成为普通拓扑命题。

随后有专门的支撑桥：

```text
Stokes/Global/CoverIndexedZeroLocalizedSupport.lean
```

关键 theorem 包括：

```lean
ManifoldForm.transitionPullbackInChartZero_localizedForm
ManifoldForm.transitionPullbackInChartZero_localizedForm_tsupport_subset_coefficient
ManifoldForm.transitionPullbackInChartZero_localizedForm_tsupport_subset_form
ManifoldForm.transitionPullbackInChartZero_localizedForm_tsupport_subset_halfSpaceSupportBox_of_support_subset_K
zero_localized_tsupport_subset_halfSpaceSupportBox_of_support_subset_K
```

这些 theorem 把如下数学逻辑形式化：

```text
support(ρ · ω) ⊆ support(ρ) ∩ support(ω)
support(ω) ⊆ K
coefficient support on K is inside coordinate box
chart-coordinate image of active carrier is controlled
⇒ zero-localized chart representative has tsupport inside selected box
```

最终 first-principles 路线使用：

```lean
CoverIndexedZeroCompactRepresentedStokesIntrinsicInput.zeroSupportOfSelectedSmoothRefinement
```

位置：

```text
Stokes/Global/CoverIndexedIntrinsicZeroSupportConstructor.lean
```

它专门针对 canonical `selectedSmoothRefinement` 证明：

```lean
tsupport
  (ManifoldForm.transitionPullbackInChartZero I
    boundaryChart boundaryChart
    (ManifoldForm.localizedForm I refinedCoefficient ω))
⊆ halfSpaceSupportBox lower upper
```

这个 theorem 是最终 represented Stokes 证明中杀人工边界的关键输入。

## ordinary support 到 tsupport 的闭集路线

另一个细节是：很多局部证明先能得到 ordinary support 包含关系，最终 half-space local Stokes 需要 topological support (`tsupport`) 或能推出人工边界 disjoint 的闭支撑信息。

这里不能随便把 `Function.support` 当成 `tsupport`。正确路线是：

```text
ordinary support subset closed box
closed half-space support box
⇒ closure support subset box
⇒ tsupport subset box
```

这就是为什么 half-space support box 的 closedness、selected box containment、open shrink support control 都需要显式证明。纸面证明里“支撑在盒子里”一句话，在 Lean 中拆成 support、closure、closed set、tsupport 的链条。

## 对主 theorem 的贡献

最终的 zero-localized support 进入：

```lean
hasIntrinsicRoute_intrinsic
```

再进入：

```lean
CoverIndexedZeroCompactRepresentedStokesIntrinsicInput.representedStokes
CoverIndexedZeroCompactRepresentedStokesFirstPrinciplesInput.representedStokes
```

它消掉的是旧路线中最不自然的支撑证书：用户不再需要证明 ordinary total chart representative 的支撑假设。Lean 内部用 canonical smooth refinement 和 zero-localization 自动生成真正需要的 half-space box support。

## 论文可用表述

中文表述：

> 形式化过程中最重要的支撑洞见是：普通 chart representative 在 Lean 中是 total function，因此纸面上的局部支撑命题不能直接照搬。我们引入 zero-localized representative，将 chart 外区域定义为零，并证明 selected smooth refinement 的 zero-localized representative 的 topological support 落入对应 half-space support box。这一桥接定理是人工边界项消失的核心输入。

英文表述：

> A naive support theorem for chart representatives is false at the level of Lean total functions. The development therefore introduces zero-localized chart representatives and proves that the representatives generated by the selected smooth refinement have topological support inside the assigned half-space boxes. This converts the informal phrase "the local form is supported in the box" into a checkable topological support theorem.
